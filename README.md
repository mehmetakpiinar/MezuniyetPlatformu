# 🎓 Mezuniyet Platformu

Bu proje, mezunlar, öğrenciler ve işverenler arasında etkileşim sağlayan bir web platformudur.

## 🚀 Özellikler

- 👤 Kullanıcı kayıt / giriş sistemi
- 💼 İş ilanı paylaşma ve başvuru
- 📝 Deneyim paylaşımı (post sistemi)
- 💬 Mesajlaşma sistemi
- 🏢 Şirket profilleri
- 🎓 Üniversite ve mezun bilgileri
- 📊 Dashboard ve istatistikler

## 🏗️ Proje Mimarisi

Bu proje katmanlı mimari (Layered Architecture) kullanılarak geliştirilmiştir:

- API → Backend servisleri (ASP.NET Core Web API)
- Business → İş mantığı
- DataAccess → Veritabanı işlemleri (Entity Framework)
- Entities → Veri modelleri
- Web → Frontend (MVC)

## 🛠️ Kullanılan Teknolojiler

- ASP.NET Core (.NET 8)
- Entity Framework Core
- SQL Server
- MVC
- RESTful API
